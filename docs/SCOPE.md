# DJ Golf Card

A simple golf scorecard Flutter app for personal use.

## Platform
- Flutter app targeting both iOS and Android simultaneously

## Courses
- Manually create courses (hole count + par per hole)
- AI-generate a course scorecard for known courses (e.g. Revere in Henderson) — populates hole count and par per hole

## Scoring
- Stroke play only
- No handicap tracking

## Players
- Multiple players per round
- Single device; one person enters all scores

## History
- Stored fully on-device (no backend, no accounts, no login)
- Per-round history includes:
  - Total score per round
  - Score per hole per round
